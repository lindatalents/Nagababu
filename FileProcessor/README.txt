1] Copy executable jar "FileProcessor-jar-with-dependencies.jar" to the suitable location.
2] Enter the below command 
    java -jar FileProcessor-jar-with-dependencies.jar names.txt asc/desc
	Provide a txt file name which want to process as as the argument and asc/desc order for sorting. for simplity make sure all the words contains only (i have given sample file names.txt)
5] click on Enter
6] log file is configured at C:\\logging.log 
	(if application gives any exception on this logger file creation such as previzies, try to run the command prompt as an adminstrator)
